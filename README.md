> git remote add


## Install

```sh
$ npm install --save git-remote-add
```

## Usage

### JavaScript

```js
var gitRemoteAdd = require('git-remote-add');

gitRemoteAdd('yashprit', 'git-remote-add', cb);
```

### Command Line reference

```sh
$ npm install --global git-remote-add
$ git-remote-add --u yashprit --r is-git 
```
